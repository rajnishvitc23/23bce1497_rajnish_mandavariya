# Real-Time Bus Tracking System

Track and monitor public buses in real time through an interactive web interface. This project uses GPS-based location updates, a backend server, and a dynamic map visualization to provide live bus positions for commuters and transport administrators. It helps improve public transportation efficiency by making bus movement transparent and easily accessible.

---

## Core Features

- Displays live bus locations on an interactive map using GPS coordinate updates.
- Tracks multiple buses simultaneously and updates their positions dynamically.
- Provides route-based visualization to help users understand bus movement along predefined routes.
- Stores and manages bus data and location updates through a backend API.
- Offers a responsive web interface accessible from desktop and mobile devices.
- Allows integration with external GPS devices or simulated location data for testing.

---

## Tech Stack

### Backend
- Node.js for server-side runtime.
- Express.js for building REST APIs and handling server logic.
- MongoDB with Mongoose for database storage and data modeling.
- Dotenv for managing environment variables.

### Frontend
- Web-based interface built with modern JavaScript technologies.
- Map visualization using Google Maps API for displaying live bus locations.

---

## Dependencies

- Node.js and npm for runtime and package management.
- MongoDB for database services.
- Additional libraries including Express, Mongoose, Dotenv, and Nodemon for development.
